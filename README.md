## Troubleshooting

### Common Issues
- **"FileNotFoundError"**: Download metadata.csv from the Kaggle link above
- **Memory Issues**: The dataset is large - ensure sufficient RAM (4GB+ recommended)
- **Slow Loading**: First run may be slow due to data processing
